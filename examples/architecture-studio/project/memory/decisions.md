# Design Decisions

## D-001: Use a sample ladder before material approval

- Projects: Project Atrium, Project Garden Flat, Project Lantern Row.
- Collaborators: Lead Designer, Materials Lead, Client Sponsor.
- Decision: compare baseline, stretch, and fallback samples in the actual room condition before approval.
- Reason: each project showed a finish that changed noticeably between showroom lighting and site lighting.
- Sensitive detail handling: exact supplier names and quotes are generalized to material categories and price bands.
- Global lesson: promoted to `global-wiki/wiki/material-samples.md`.

## D-002: Favor warm, low-glare lighting in transition spaces

- Projects: Project Atrium, Project Lantern Row.
- Collaborators: Lighting Consultant, Lead Designer.
- Decision: use warm, low-glare fixtures in reception, entry, and corridor zones.
- Reason: both spaces need wayfinding without making narrow areas feel clinical.
- Sensitive detail handling: site photos and exact fixture quote are blocked; the reusable rule is kept.
- Global lesson: queued as a lighting checklist after one more project confirms the pattern.

## D-003: Generalize private routines into traffic patterns

- Projects: Project Garden Flat, Project Lantern Row.
- Collaborators: Lead Designer, Site Coordinator.
- Decision: document "morning traffic" and "evening use" instead of personal schedules.
- Reason: the design needs circulation and storage context, not private household details.
- Sensitive detail handling: original routine notes are replaced with privacy-safe usage windows.
- Global lesson: keep as a local privacy pattern until repeated by another team.

## D-004: Use acoustic texture where hard finishes dominate

- Projects: Project Atrium, Project Garden Flat.
- Collaborators: Materials Lead, Lighting Consultant.
- Decision: balance hard surfaces with acoustic or tactile texture when the palette uses stone-look counters, glass, or sealed floors.
- Reason: both projects felt sharper and louder after hard finish options were added.
- Sensitive detail handling: private room photos are not stored; only material behavior is recorded.
- Global lesson: add to a future finishes checklist if another project repeats it.
